如果日期格式集错误，可以按如下命令操作进行恢复：

（1）sqlplus /nolog

（2）SQL> connect sys/oracle@minos as sysdba;

（3）SQL> SHUTDOWN immediate; 关闭数据库

（4）SQL> CREATE pfile FROM spfile; 导出pfile

     此命令将在缺省目录创建文件 init${ORACLESID}.ora.
		 
    UNIX下的目录：
		
    ${ORACLE_HOME}/dbs/ 或
		
    windows下的目录：
		
    ${ORACLE_HOME}\database

（5）修改init${ORACLESID}.ora文件添加一行

*.nls_date_format = 'YYYY-MM-DD HH24:MI:SS'

（6）SQL> CREATE spfile FROM pfile; 将修改导入到spfile

（7）SQL>startup; 重新加载数据库
